Sobre mí

Soy Andrés Martínez, estudiante de Ingeniería Informática en la Universidad de Murcia, donde estoy finalizando mi tercer año (2022 - presente). NanoFiles

NanoFiles es un sistema de compartición de archivos peer-to-peer desarrollado en Java, diseñado para implementar protocolos de red personalizados sobre UDP y TCP. Este proyecto incluye un servidor de directorio y clientes/servidores, ofreciendo una solución robusta para la transferencia de archivos entre pares.

Características implementadas:

  • Protocolo confiable sobre UDP para comunicación con el servidor de directorio.

  • Funcionalidades básicas: inicio de sesión, listado de usuarios, publicación y búsqueda de archivos, descargas y cierre de sesión.
  • Mejoras opcionales: soporte para servidores multi-hilo y descargas paralelas desde múltiples servidores.

Este proyecto demuestra mi experiencia en diseño e implementación de protocolos de red, así como en programación concurrente y sistemas distribuidos.

Tecnologías utilizadas: Java.

Compilador en Python

Este proyecto consiste en la creación de un compilador en Python utilizando autómatas lineales formales para el análisis léxico y sintáctico. El compilador procesa un lenguaje de programación personalizado, generando código intermedio que puede ser ejecutado en una máquina virtual sencilla. El proyecto abarca todas las fases de un compilador, desde el análisis léxico hasta la generación de código.

Características implementadas:

  • Análisis léxico mediante autómatas finitos deterministas (DFA) para tokenización.
  • Análisis sintáctico con un parser descendente recursivo para validar la gramática.
  • Generación de código intermedio en un formato simplificado (bytecode).
  • Manejo de errores con mensajes detallados para el usuario.
  • Optimización básica del código generado, eliminando redundancias simples.

Este proyecto refleja mi capacidad para aplicar conceptos teóricos de teoría de autómatas y compiladores en un entorno práctico, así como mi experiencia con Python para desarrollar herramientas complejas.

Tecnologías utilizadas: Python.
